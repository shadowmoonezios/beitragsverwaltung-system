# Beitragsverwaltungssystem

## Übersicht
Das Beitragsverwaltungssystem ist eine Full-Stack-Anwendung, die es gemeinnützigen Organisationen ermöglicht, ihre finanziellen Aktivitäten effektiv zu verwalten. Die Anwendung bietet eine benutzerfreundliche Oberfläche zur Verwaltung von Spenden, Ausgaben und zur Erstellung von Finanzberichten.

## Funktionen
- **Spendenverwaltung:** Verfolgen von eingegangenen Spenden und Spenderinformationen.
- **Ausgabenmanagement:** Nachverfolgung der Ausgaben und Kategorisierung.
- **Berichterstattung:** Erstellen von Finanzberichten und -statistiken.
- **Benutzerverwaltung:** Verwaltung von Benutzern mit unterschiedlichen Berechtigungen.
- **Benachrichtigungen:** E-Mail-Benachrichtigungen über wichtige Ereignisse und Meilensteine.

## Technologien
- Frontend: React.js
- Backend: Node.js mit Express
- Datenbank: MongoDB
- Authentifizierung: JWT (JSON Web Token)

## Installation
1. Klone das Repository.
2. Installiere die Abhängigkeiten mit `npm install` im Hauptverzeichnis und im Client-Verzeichnis.
3. Starte die Datenbank und konfiguriere die Umgebungsvariablen.
4. Führe die App mit `npm start` aus.
